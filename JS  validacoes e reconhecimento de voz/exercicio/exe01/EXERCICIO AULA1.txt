1) Criando e integrando módulos JavaScript
Imagine que você está desenvolvendo um jogo de adivinhação online, em que os jogadores precisam adivinhar um número secreto gerado aleatoriamente. Para organizar o projeto e facilitar sua manutenção, você escolhe dividir o código JavaScript em módulos separados. Sua tarefa é: crie um módulo chamado sortearNumero.js, responsável por gerar e exibir um número secreto aleatório. Além disso, você deve integrar este módulo ao seu projeto HTML principal. Siga as instruções:

Crie uma pasta chamada “app” no seu projeto para armazenar todos os arquivos JavaScript;
Dentro da pasta “app”, crie um arquivo chamado sortearNumero.js;
No arquivo sortearNumero.js, escreva uma função que gera um número aleatório entre 0 e 99 e exiba esse número no console usando o console.log;
Integre o módulo sortearNumero.js ao seu arquivo HTML principal (index.html), adicionando a tag <script src="app/sortearNumero.js"></script> na parte inferior do body;
Teste o seu projeto, abrindo o arquivo index.html no navegador e verifique se o número aleatório é exibido no console ao recarregar a página.
2) Criando um sorteio de números aleatórios
Continuando o projeto do jogo, uma parte essencial é o sorteio de um número secreto, que deve ser completamente aleatório para garantir a imparcialidade. Sua tarefa é: implemente essa funcionalidade utilizando JavaScript. Siga as instruções:

Crie um arquivo chamado sortearNumero.js;
Dentro do arquivo criado, escreva uma função que sorteia um número aleatório entre 0 e 100, inclusive;
Mostre esse número no console com a mensagem "Número Secreto: [número]". Lembre-se de incluir este script no seu arquivo index.html para testar o resultado.
