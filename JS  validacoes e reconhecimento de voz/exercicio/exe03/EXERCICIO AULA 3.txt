PRÓXIMA ATIVIDADE

Vamos praticar o que aprendemos até aqui?

1) Integrando reconhecimento de voz ao seu projeto
Você está desenvolvendo um jogo de adivinhação e decidiu torná-lo mais interativo, permitindo que os usuários possam falar os palpites em vez de digitá-los. Utilize a Web Speech API para adicionar reconhecimento de voz ao seu projeto. Siga as instruções a seguir para fazer o exercício:

Crie um novo arquivo chamado reconhecimentoDeVoz.js. Neste arquivo, configure o reconhecimento de voz para ser ativado assim que a aplicação for carregada, e ajuste-o para reconhecer a língua portuguesa;
Teste se o reconhecimento de voz está funcionando, verificando se o microfone do navegador é ativado ao recarregar a página.
2) Exibindo a fala reconhecida no console
Agora que o microfone pode ser ativado, o próximo passo é capturar o que é falado e exibir essa informação.

Modifique o arquivo reconhecimentoDeVoz.js para que, além de iniciar o reconhecimento de voz, ele também possa capturar a fala do usuário e exibi-la no console;
Implemente um evento que seja disparado toda vez que o usuário falar algo, capturando essa fala e exibindo-a no console como "Você disse: [frase]". Isso permitirá que você veja em tempo real o que o programa está reconhecendo.
3) Exibindo chutes na interface do usuário
No jogo de adivinhação, desenvolva uma funcionalidade que permite aos jogadores fazerem chutes, usando a voz. Quando um jogador fala um número, esse número deve ser capturado pelo sistema e exibido na interface do jogo, especificamente em uma área designada para mostrar o chute do jogador. Para fazer isso, utilize o reconhecimento de voz para capturar o chute e o JavaScript para atualizar o conteúdo de uma div com o ID "chute", exibindo o texto "Você disse: [número]" junto com o número falado.

Bom trabalho!
